API REST - Gonzalo Gonzalez

Endpoints Users ▶

GET /api/users ▶
Obtiene la lista de todos los usuarios.

Respuesta exitosa (200 OK)

[
    {
        "id": 1,
        "name": "Gonzalo",
        "email": "gonzalo@example.com",
        "age": 30
    }
]

Errores comunes

  • 500 Internal Server Error: Problema en el servidor.
GET /api/users/:id ▶
Obtiene un usuario por su ID.

Parámetros

  • id (path): ID del usuario (número entero).

Respuesta exitosa (200 OK)

{
    "id": 1,
    "name": "Gonzalo",
    "email": "gonzalo@example.com",
    "age": 30
}

Errores comunes

  • 404 Not Found: Usuario no encontrado.
  • 400 Bad Request: ID inválido.
POST /api/users ▶
Crea un nuevo usuario.

Body (JSON)

{
    "name": "Nuevo Usuario",
    "email": "nuevo@example.com",
    "age": 25
}

Respuesta exitosa (201 Created)

{
    "id": 2,
    "name": "Nuevo Usuario",
    "email": "nuevo@example.com",
    "age": 25
}

Errores comunes

  • 400 Bad Request: Datos faltantes o inválidos.
  • 409 Conflict: Email ya registrado.
PUT /api/users/:id ▶
Actualiza un usuario existente.

Parámetros

  • id (path): ID del usuario a actualizar.

Body (JSON)

{
    "name": "Gonzalo Actualizado",
    "email": "gonzalo.upd@example.com",
    "age": 31
}

Respuesta exitosa (200 OK)

{
    "id": 1,
    "name": "Gonzalo Actualizado",
    "email": "gonzalo.upd@example.com",
    "age": 31
}

Errores comunes

  • 400 Bad Request: Datos inválidos.
  • 404 Not Found: Usuario no encontrado.
DELETE /api/users/:id ▶
Elimina un usuario por su ID.

Parámetros

  • id (path): ID del usuario a eliminar.

Respuesta exitosa (204 No Content)

El cuerpo de la respuesta está vacío.

Errores comunes

  • 404 Not Found: Usuario no encontrado.
  • 400 Bad Request: ID inválido.
